# Weaving Punch Card Calculator

A web-based tool designed for the textile weaving industry to automate the creation of punching card descriptions from ArahPaint design images.

## Features
- Upload ArahPaint designs (BMP/PNG)
- Automatically identify pattern colors
- Generate precise punching card notation formatted into ranges
- Glassmorphism dark-theme UI with an interactive grid canvas
- Export outputs to TXT or DOCX formats

## Usage
Simply open `index.html` in your web browser. Upload an image, configure your thread and board counts, map your colors, and generate the output. No installation or server required.
