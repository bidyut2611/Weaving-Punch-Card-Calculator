❓ How to Use

📊 Punch Card Calculator

  1. Open your design in ArahPaint and export/save it as a BMP or PNG file.
  2. Upload the image by clicking the upload area or dragging the file.
  3. Set the configuration:
    • Design Name — e.g., "Buta 50"
    • Thread (Width) — number of columns in the grid (e.g., 100)
    • Board (Height) — number of rows in the grid (e.g., 50)
  4. Click "Process Design" to analyze the image.
  5. Review the Detected Colors panel:
    • Set each color as Primary (main design) or Mina (secondary color section).
    • For Mina colors, you can set a custom label.
    • Set to Ignore to exclude a color.
  6. Export the result as TXT or DOCX file.
💡 Notation Format
50, 51 → Individual columns
48--53 → Three or more consecutive columns
|| → End of row marker

🔄 Image → BMP Converter

  1. Switch to the "Image → BMP" tab in the header.
  2. Upload any PNG, JPG, or other image of your design idea.
  3. Set the Width and Height in pixels (matching your Thread × Board count).
  4. Choose Max Colors to control the palette (e.g., 2 for black & white).
  5. Click "Convert to BMP" to generate the textile-ready BMP.
  6. Download the BMP, or click "Use in Punch Calculator" to directly send it to the calculator for further processing.
